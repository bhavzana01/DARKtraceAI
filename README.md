# DARKtraceAI
AI-powered Insider Threat Detection and Privileged Access Protection System.

## Problem

Privileged accounts in banking systems can be misused by employees,
vendors, contractors, or attackers.

## Solution

SecureGuard AI combines:

- AI Behaviour Analysis
- Risk-Based Authentication
- Biometric Verification
- Real-time Threat Detection

## Features

- Detect unusual user behaviour
- Generate risk scores
- Protect privileged accounts
- Automated security alerts
